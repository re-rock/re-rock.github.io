Slim
a micro framework for PHP

Slimは、シンプルかつパワフルなWebアプリケーションとAPIをすばやく作成することができるPHPマイクロフレームワークです。 

            <?php
                use Psr\Http\Message\ResponseInterface as Response;
                use Psr\Http\Message\ServerRequestInterface as Request;
                use Slim\Factory\AppFactory;

                require __DIR__ . '/../vendor/autoload.php';

                $app = AppFactory::create();

                $app->get('/hello/{name}', function (Request $request, Response $response, array $args) {
                    $name = $args['name'];
                    $response->getBody()->write("Hello, $name");
                    return $response;
                });

                $app->run();

Download & Install

SlimフレームワークはComposer を利用してインストールすることをお勧めします。

Slimで作業を開始する最も簡単な方法は、次のコマンドを実行しSlim-Skeleton をベースとしたプロジェクトを作成することです。

$ php composer.phar create-project slim/slim-skeleton [my-app-name]

[my-app-name]の部分をあなたが作成するアプリケーションの任意のディレクトリ名に置き換えてください。

作成したアプリケーションはPHPのビルトインサーバーで実行できます。

$ cd [my-app-name]; php -S localhost:8080 -t public public/index.php

特徴

HTTP Router

Slimは特定のHTTPリクエストのルートコールバックとURIを紐付ける早くて協力なルーターを提供します。 またパラメータとパターンマッチングをサポートしています。

Middleware

アプリのビルド時にSlimアプリのHTTPリクエストとレスポンスオブジェクトを調整する核となるミドルウェアが作成されます。

PSR-7 Support

SlimはPSR-7メッセージをサポートしているため、HTTPメッセージメソッド、ステータス、URI、ヘッダー、クッキー、そしてボディを検証および操作できます。

Dependency Injection

Slimは「依存性の注入」をサポートしています。そのため外部ツールを完全にコントロールでき、どんなコンテナ相互運用も利用できます。
